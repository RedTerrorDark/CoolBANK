# Банкомат
## Описание
Это простой проект имитации банкомата обладающий возможностью создания и входа в аккаунты, перевод виртуальной "валюты" на разные аккаунты, внесение и снятие виртуальной валюты с аккаунта.
### Задача:
Создать имитацию банкомата с всеми функциями реального банкомата.
### Используемый паттерн:
Singleton - идеально подойдёт для объекта, который должен быть единым источником истины. В данном случае Singleton используется для базы данных. Ведь если создать несколько экземпляров базы данных в разных частях программы, можно получить рассинхронизацию балансов, дублирование транзакций или утечку памяти. Singleton решает это архитектурно.
